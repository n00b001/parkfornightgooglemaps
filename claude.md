# Park4Night Google Maps — Project Instructions

## Tech Stack
- **Client**: React + TypeScript + Vite + Tailwind CSS + @react-google-maps/api
- **Server**: Node.js + Express + Passport (Google OAuth 2.0) + Prisma + PostgreSQL
- **Deployment**: Render (static client + Node.js server + managed PostgreSQL)

## Architecture
- Monorepo: `client/` (React SPA), `server/` (Express API)
- Deployed via `render.yaml` — client at `park4night-client.onrender.com`, server at `park4night-server.onrender.com`
- Google OAuth flow: client redirects to `/auth/google` on server → Google consent → `/auth/google/callback` → redirect back to client
- Session stored in PostgreSQL via `connect-pg-simple`

## Key Environment Variables
| Variable | Service | Purpose |
|---|---|---|
| `GOOGLE_CLIENT_ID` | server | Google OAuth client ID |
| `GOOGLE_CLIENT_SECRET` | server | Google OAuth client secret |
| `CLIENT_URL` | server | Client origin for CORS & OAuth redirect (auto-set by Render) |
| `SERVER_URL` | server | Server origin for OAuth callback URL (auto-set by Render) |
| `DATABASE_URL` | server | PostgreSQL connection string (auto-set by Render) |
| `SESSION_SECRET` | server | Session signing secret (auto-generated by Render) |
| `VITE_API_URL` | client | Server host for API calls (auto-set by Render) |
| `VITE_GOOGLE_MAPS_API_KEY` | client | Google Maps JavaScript API key |

## Google Cloud Console Configuration
The OAuth 2.0 client must have:
- **Authorized redirect URIs**: `https://park4night-server.onrender.com/auth/google/callback`
- **Authorized JavaScript origins**: `https://park4night-client.onrender.com`

## Development
```bash
npm install
cd server && npm install && npx prisma generate
# Start server: cd server && npm start
# Start client: cd client && npm run dev
```
